# Guía de Respuesta a Incidentes de Seguridad

Bienvenido al repositorio de procedimientos, formularios y guías rápidas para la gestión y respuesta a incidentes de seguridad informática.

## Introducción

Este repositorio contiene la documentación y herramientas necesarias para el manejo efectivo de incidentes de seguridad en la organización. Aquí encontrarás procedimientos detallados, formularios de registro, guías rápidas y scripts de apoyo para el equipo de seguridad y TI.

## Contenido principal

- **[Procedimiento de Respuesta a Incidentes](procedimiento_respuesta_incidentes.md):**
  Documento principal con el paso a paso para la gestión de incidentes, desde la detección hasta la remediación y lecciones aprendidas.

- **[Guía Rápida de Respuesta a Incidentes](guia_rapida_respuesta_incidentes.md):**
  Cheat sheet para el equipo de seguridad con comandos, acciones inmediatas y checklist mínimo.

- **[Formulario de Registro de Incidente](formulario_registro_incidente.md):**
  Formato estructurado para documentar cada incidente, sus acciones y resultados.

- **[script/CollectIncidentEvidence.ps1](script/CollectIncidentEvidence.ps1):**
  Script PowerShell para recolección rápida de evidencia en equipos Windows.

## ¿Cómo usar este repositorio?

1. Consulta el **Procedimiento de Respuesta a Incidentes** para el flujo completo y roles.
2. Utiliza la **Guía Rápida** para acciones inmediatas y comandos forenses.
3. Registra cada incidente usando el **Formulario**.
4. Apóyate en los scripts para la recolección de evidencia técnica.
